Welcome to Skillbox — the missing infrastructure piece for AI agents that need to execute code safely.

## What is Skillbox?

Skillbox is a **self-hosted execution runtime** that lets your AI agents run Python, Node.js, and Bash scripts in hardened sandboxes. Think of it as a package registry for agent capabilities — skills are versioned, discoverable, and return structured JSON.
